## What is BMAD Avanade?

**BMAD** (Business-Minded AI Development) is an open-source AI-powered development methodology that combines structured role-based AI agents with a proven software delivery workflow. **BMAD Avanade** extends the core BMAD method with Avanade-specific best practices, templates, and integrations — tailored to Avanade's ways of working and technology landscape.

Each module is a self-contained extension that plugs into the BMAD agent framework, giving your AI copilot deep expertise in a specific domain.

> **Note:** The source repositories for each module are internal to Avanade. Avanade employees with the appropriate access can follow the repository links to view the full source code, contribute, and raise issues.
